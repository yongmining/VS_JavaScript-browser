02. HTML 어트리뷰트와 DOM 프로퍼티

HTML 어트리뷰트 : HTML 요소의 초기 상태를 지정하며 변하지 않는다. 어트리뷰트 노드에서 관리되며 값을 얻거나 변경할 때 getAttribute/setAttribute 메소드를 이용한다. DOM 프로퍼티 : 사용자가 입력한 최신 상태를 관리한다. 사용자의 입력에 의한 상태 변화에 반응하여 언제나 최신 상태를 유지한다.


getAttribute메소드로 취득한 어트리뷰트 값은 언제나 문자열이다. 하지만 DOM 프로퍼티로 취득한 최신 상태 값은 문자열이 아닐 수 있다.
예) checkbox 요소의 checked 어트리뷰트 값은 문자열이지만 checked 프로퍼티 값은 불리언 타입이다.
